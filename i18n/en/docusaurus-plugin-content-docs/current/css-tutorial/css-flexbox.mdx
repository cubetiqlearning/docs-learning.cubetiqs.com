# CSS - Flexbox

### What is CSS Flexbox?

- **CSS Flexbox** is used to define each element to be beautiful and easy to position. In this Flexbox, we have two things to study: `Parent Element` and `Child Element`.

### ចំណុចពិសេសរបស់ CSS Flexbox

- **Flexbox** នៅក្នុង **CSS** គឺអាចធ្វើការបែងចែក `Container` នៅក្នុង គេហទំព័ររបស់អ្នកឲ្យមានលក្ខណះស្តង់ដា រួមបញ្ចូលជាមួយនឹង `Properties` ជាច្រើន ដែលផ្តល់ភាពងាយស្រួលដល់អ្នកនៅពេលដែលអ្នកត្រូវការ ហើយវាក៏ផ្នែកមួយដ៏សំខាន់ក្នុងការធ្វើ `Responsive` ទៅលើគេហទំព័រ។

### Properties របស់ CSS Flexbox

**1. Parent Element**:

- នៅក្នុង `Parent Element` នេះគឺមាន `properties` ជាច្រើនដែលយើងត្រូវសិក្សា ដែលមានដូចខាងក្រោម:

  - **flex-direction**: ប្រើសម្រាប់កំណត់ទិសដៅនៃធាតុដែលវាផ្ទុក។

  - **flex-wrap**: ប្រើសម្រាប់កំណត់ថាតើធាតុដែលវាផ្ទុកគួរចុះបន្ទាត់ឬអត់ នៅពេលដល់កន្លែងទាល់។

  - **flex-flow**: ប្រើសម្រាប់សរសេរតែក្នុងមួយ `property` រវាង `flex-direction` និង `flex-wrap` ។

  - **justify-content**: ប្រើសម្រាប់តម្រៀបធាតុទៅតាមទិសដៅនៃទិសដៅគោល។

  - **align-content**: ប្រើសម្រាប់តម្រៀបធាតុទៅតាមទិសដៅដែលកាត់ទិសដៅគោល(សម្រាប់ច្រើនបន្ទាត់)។

  - **place-content**: ប្រើសម្រាប់សរសេរតែក្នុងមួយ `properties` រវាង `justify-content` និង `align-content` ។

  - **align-items**: ប្រើសម្រាប់តម្រៀបទៅតាមទិសដៅដែលកាត់ទិសដៅគោល(សម្រាប់មួយបន្ទាត់) ។

**2. Child Element**:

- នៅក្នុង `Child Element` នេះគឺមាន `properties` ជាច្រើនដែលយើងត្រូវសិក្សា ដែលមានដូចខាងក្រោម:

  - **order**: ប្រើសម្រាប់កំណត់លំដាប់។

  - **flex-grow**: ប្រើសម្រាប់ពង្រីកទំហំទៅឲ្យធាតុនូវទំហំដែលនៅសល់។

  - **flex-shrink**: ប្រើសម្រាប់ពង្រួមធាតុ។

  - **flex-basis**: ប្រើសម្រាប់កំណត់ទំហំទៅឲ្យធាតុ។

  - **flex**: ប្រើសម្រាប់សរសេរតែក្នុងមួយ `properties` រវាង `flex-grow`, `flex-shrink` និង `flex-basis` ។

  - **align-self**: ប្រើសម្រាប់កំណត់តម្លៃដូច `align-items` ប៉ុន្តែសម្រាប់តែមួយធាតុបច្ចុប្បន្ននេះទេ។

### ឧទាហរណ៍:

import CssFlexBox from "@site/src/components/CssFlexBoxFeatures";

`index.html`

```html
<div class="container">
  <div class="content content__one">
    <div class="box box__one">1</div>
    <div class="box box__two">2</div>
    <div class="box box__three">3</div>
    <div class="box box__four">4</div>
    <div class="box box__five">5</div>
  </div>
</div>
```

`style.css`

```css
.container {
  width: 100%;
  position: relative;
}

.content {
  width: 90%;
  height: 360px;
  margin: 30px auto;
  background-color: #6c3483;
  border-radius: 10px;
  border: 2px solid #f39c12;
  padding: 10px;
}

.box {
  margin: 10px;
  width: 45%;
  height: 45px;
  border-radius: 10px;
  border: 2px solid #dedede;
}

.box__one {
  background-color: #2ecc71;
}

.box__two {
  background-color: #e74c3c;
}

.box__three {
  background-color: #5499c7;
}

.box__four {
  background-color: #f5b041;
}

.box__five {
  background-color: #85929e;
}
```

`Result:`

<CssFlexBox />
