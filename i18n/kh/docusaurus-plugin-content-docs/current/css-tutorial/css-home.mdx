# CSS - ទំព័រដើម

### អ្វីទៅជា CSS?

- **CSS** គឺត្រូវបានប្រើសម្រាប់រចនានូវអ្វីគ្រប់យ៉ាងនៅលើគេហទំព័រដូចជាការកំណត់ពណ៌លើអក្សរ, កំណត់ពណ៌នៃផ្ទៃខាងក្រោយ, កំណត់ទំហំអក្សរ, កំណត់គម្លាត រួមទាំងមានការរចនាប្លង់ និងការប្រែប្រួលនៃការបង្ហាញសម្រាប់ឧបករណ៍និងទំហំអេក្រង់ផ្សេងៗគ្នា(`Responsive`)។

### តើ CSS មានពាក្យពេញជាអ្វីវិញ?

- **CSS** មានពាក្យពេញគឺ **Cascading Style Sheets**។

### ហេតុអ្វីបានជាត្រូវប្រើ CSS?

- **CSS** ត្រូវបានប្រើដើម្បីកំណត់រចនាស្ទាយសម្រាប់គេហទំព័ររបស់អ្នក រួមទាំងការរចនា ប្លង់ និងបំរែបំរួលក្នុងការបង្ហាញសម្រាប់ឧបករណ៍ និងទំហំអេក្រង់ផ្សេងៗគ្នា។

### តម្រូវការមុននឹងសិក្សា CSS

- មុននឹងអ្នកអាចសិក្សាមេរៀន **CSS** នេះបាន អ្នកត្រូវចេះនូវមេរៀន **HTML** ជាមុនសិន។
