# Spring Boot Kotlin - ការ Bootstrapping

ជំពូកនេះនឹងពន្យល់អ្នកពីរបៀបក្នុងការ bootstrapping នៅលើកម្មវិធី Spring Boot ។

### Spring Initializer

- ដើម្បីអាចបង្កើតនូវ **Project Spring Boot** មួយបានគឺមានវិធីច្រើន តែនៅក្នុង **Tutorial** មួយនេះគឺយើងនឹងនាំអ្នកទៅបង្កើត **Project Spring Boot** នៅលើគេហទំព័រ <a href="https://start.spring.io/" target="_blank">Spring Initializer</a> ។

![Spring Initializer Dashboard](/img/docs/start-spring-io.jpg)

- បន្ទាប់ពីបានរើសនូវ(*Project ជា Gradle Project, Language ជា Kotlin, Spring Boot Version ចុងក្រោយ, ផ្តល់ជា Project Metadata ដូចជា Group, Artifact, Name, Description, Package name, Packaging ជា Jar, Java Version ដែលអ្នកចង់ប្រើ, Dependencies គឺរើសយក Spring Web សម្រាប់បង្កើតជា API និង Web MVC រីឯ Spring Data JPA សម្រាប់ភ្ជាប់ទិន្នន័យនៅក្នុង SQL ជាមួយនឹង Java Persistence API ដោយប្រើប្រាស់ Spring Data និង Hibernate រីឯ H2 Database ជាប្រភេទ in-memory database*) បានរួចរាល់ហើយប៊ូតុង **Generate** ដើម្បី **Generate** នូវ **Project** ។
