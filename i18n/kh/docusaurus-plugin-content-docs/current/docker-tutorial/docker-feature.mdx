# Docker - Features

មុខងារសំខាន់ៗមួយចំនួនរបស់ **Docker** មានដូចខាងក្រោម៖

- Easy and Faster configuration
- Increase productivity
- Application Isolation
- Swarm
- Routing Mesh
- Services
- Security Management

### Easy and Faster configuration

- នេះជាមុខងារសំខាន់របស់ **Docker** ដែលអាចជួយយើងក្នុងការកំណត់នូវរចនាសម្ព័ន្ធបានយ៉ាងងាយស្រួលនិងលឿនជាងមុន។

- យើងអាចធ្វើការ `deploy` នូវកូដរបស់យើងដោយចំណាយពេលតិច។

### Increase productivity

- **Docker** បន្ធូរបន្ថយទាក់ទងនឹងបច្ចេកទេសនៃការកំណត់រចនាសម្ព័ន្ធនិងការ `deploy` នូវកម្មវិធីមានភាពឆាប់រហ័ស។

### Application Isolation

- វាបានផ្តល់នូវ `container` ដែលប្រើសម្រាប់ដំណើរការនូវកម្មវិធីដោយមានលក្ខណៈដាច់ដោយឡែក។

### Swarm

- វាជា `clustering` និង `scheduling` សម្រាប់ **Docker Containers**។ វាអនុញ្ញាតឲ្យអ្នកប្រើប្រាស់អាចគ្រប់គ្រងនូវ `containers` ជាច្រើនដែលបាន `deploy` នៅលើម៉ាស៊ីនជាច្រើន។

### Routing Mesh

- វាបញ្ជូនសំណើរដើម្បីអាច `published ports` នៅលើ `nodes` ដែលមាន `container` កំពុងតែ `active`។

### Services

- **Services** ជាបញ្ជីនៃ `tasks` ដែលអនុញ្ញាតឲ្យយើងអាចបញ្ជាក់ពីស្ថានភាពនៃ `container` នៅក្នុង `cluster`។

### Security Management

- វាអនុញ្ញាតឲ្យយើងអាចរក្សាទុកនូវអាថ៌កំបាំងចូលទៅក្នុង `swarm` របស់ខ្លួនវា ហើយអាចឲ្យ `services` ចូលទៅប្រើប្រាស់នូវអាថ៌កំបាំងនោះ។

- វារួមបញ្ចូលនូវពាក្យបញ្ជាសំខាន់ៗមួយចំនួនចំពោះម៉ាស៊ីនដូចជា `secret inspect`, `secret create`។ល។
